## Good First Issues for Contributors

We have a list of tasks that are great for new contributors. Check them out on our [GitHub Issues page](https://github.com/UtkarshKarale/hansa-ai/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22).

Some examples include:
*   Improve CLI error messages
*   Add support for a new programming language (e.g., Go, Rust)
*   Add more output formats (e.g., JSON, HTML)
*   Improve test coverage for existing commands
*   Refactor the folder structure for better organization

## How to Suggest a Feature

Have an idea for a new feature? Please [open an issue](https://github.com/UtkarshKarale/hansa-ai/issues/new) to discuss it. We'd love to hear your feedback!
